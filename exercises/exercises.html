Exercise 1: Dropping rows with complex conditions

This small sample mirrors the animation from the presentation. It includes an NA in body_mass_g that makes negation-based filtering tricky.

penguins_small <- tibble(
  species     = rep("Adelie", 5),
  island      = c("Torgersen", "Biscoe", "Dream", "Torgersen", "Torgersen"),
  body_mass_g = c(3750, 3800, 3300, 4500, NA_real_)
)

penguins_small
# A tibble: 5 × 3
  species island    body_mass_g
  <chr>   <chr>           <dbl>
1 Adelie  Torgersen        3750
2 Adelie  Biscoe           3800
3 Adelie  Dream            3300
4 Adelie  Torgersen        4500
5 Adelie  Torgersen          NA

Task: Drop rows where both island == "Torgersen" and body_mass_g > 4000 are true. Rows where either condition is missing (NA) should be kept.

Expected rows remaining after the operation:

species island body_mass_g Why kept
Adelie Biscoe 3800 Island ≠ Torgersen
Adelie Dream 3300 Island ≠ Torgersen
Adelie Torgersen NA body_mass_g is NA — can’t confirm it’s > 4000, keep it 
penguins_small |>
  filter(
    !((island == "Torgersen" & !is.na(island)) &
        (body_mass_g > 4000 & !is.na(body_mass_g)))
  )
# A tibble: 4 × 3
  species island    body_mass_g

1 Adelie  Torgersen        3750

4 Adelie  Torgersen          NA

Mapping:

Task: Add an island_location column using this mapping.

penguins_islands |>
  mutate(
    island_location = case_when(
      island == "Biscoe"    ~ "Southwest",
      island == "Dream"     ~ "Northwest",
      island == "Torgersen" ~ "Northwest",
      .default = island
    )
  )

Task: Add island_location using island_lookup as the mapping source.

# Manual: pull each pair out of the lookup table one-by-one
penguins_islands |>
  mutate(
    island_location = case_when(
      island == island_lookup$from[island_lookup$from == "Biscoe"]    ~
        island_lookup$to[island_lookup$from == "Biscoe"],
      island == island_lookup$from[island_lookup$from == "Dream"]     ~
        island_lookup$to[island_lookup$from == "Dream"],
      island == island_lookup$from[island_lookup$from == "Torgersen"] ~
        island_lookup$to[island_lookup$from == "Torgersen"],
      .default = island
    )
  )
